Weapon Generator API - Help page

This page has three endpoints :

  • /assault_rifle
  • /shotgun
  • /sniper_rifle

Wanna generate more ? just add a number after the endpoint !

i.e. Generate 20 sniper rifles !, Generate 20 assault rifles ! or even Generate 20 shotguns !

those three pages will return a json object. They all use the get method.

That's nice and all, but what's the point ?

This project is mostly a POC, weapon generation on the fly can be useful for an MMORPG or a looter shooter kind of game.
